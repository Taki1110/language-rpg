# 語源の旅人 (Gogen no Tabibito)

英語と中国語を同時に学べるブラウザベースのテキストRPGゲーム

![Game Screenshot](screenshot.png)

## 🎮 ゲームについて

「語源の旅人」は、言語学習とRPGゲームを融合させた教育的なゲームです。日本語を母語とするプレイヤーが、英語と中国語を同時に学びながら冒険します。

### 特徴

- 🇯🇵🇬🇧🇨🇳 **三カ国語対応**: 日本語・英語・中国語（拼音付き）を同時表示
- 📚 **言語別経験値**: 英語と中国語のスキルを個別にレベルアップ
- ⚔️ **クイズバトル**: 言語問題に答えて敵を倒す
- 🗺️ **複数エリア**: 始まりの村、英語の森、漢字の山など
- 📱 **レスポンシブ対応**: スマートフォンでもプレイ可能

## 🎯 ゲームシステム

### コマンド

| コマンド | 説明 |
|---------|------|
| 🔍 探索 | アイテムを見つけて経験値獲得 |
| 📚 勉強 | ランダムに3つの単語を学習 |
| ⚔️ 戦闘 | モンスターとクイズバトル |
| 🚶 移動 | 別のエリアへ移動 |
| 📊 ステータス | 能力値確認 |
| 📖 単語帳 | 学習した単語を復習 |

### ステータス

- **❤️ HP**: 戦闘で減少、0になるとゲームオーバー
- **⭐ Level**: 経験値でレベルアップ
- **💰 Gold**: アイテム購入に使用
- **🇬🇧 English Exp**: 英語スキル
- **🇨🇳 Chinese Exp**: 中国語スキル

### エリア

| エリア | 説明 | 敵 |
|-------|------|-----|
| 始まりの村 | チュートリアルエリア | 迷子のスライム |
| 英語の森 | 英語の魔力が満ちた森 | 文法ゴブリン、単語の狼 |
| 漢字の山 | 古代の漢字が眠る山 | 難読ドラゴン、四字熟語ゴーレム |
