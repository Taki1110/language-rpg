# 語源の旅人 - Etymology Traveler

言葉の力が魔法となる世界で、英語と中国語を学びながら冒険するRPGゲーム。

## 🎮 ゲーム概要

「語源の旅人」は、言葉の力が魔力となる世界を舞台にした英語・中国語学習RPGです。
バトルで単語クイズに答え、正解すると攻撃が成功する独自のシステムで、楽しみながら語学力を鍛えられます。

### 主な特徴

- **500語以上**の単語データベース（初級200語、中級200語、上級100語）
- **15エリア**以上の探索可能なマップ
- **30種類以上**の敵キャラクター
- **5体**のボス（パターンチェンジあり）
- **12章**からなるメインストーリー
- **25個以上**のサブクエスト
- **4つのクラス**（語学戦士、文字術師、語り部、言語学者）
- **スキルツリー**システム
- **クラフト/合成**システム

## 🚀 遊び方

### ブラウザでプレイ

1. リポジトリをクローンまたはダウンロード
2. `index.html` をブラウザで開く
3. 「新規ゲーム」を選択して冒険開始！

### 基本的な操作

- **探索する**: 現在のエリアを探索し、アイテムを見つけたり敵と遭遇したり
- **戦闘を始める**: 敵と戦闘を開始
- **話す**: NPCと会話してクエストを受ける
- **移動**: 北/南/東/西のボタンでエリア間を移動

### バトルシステム

1. 「英語攻撃」または「中国語攻撃」を選択
2. 表示された単語の日本語訳を選択
3. 正解で攻撃成功！
4. 敵の弱点を突くと追加ダメージ

## 🌍 世界観

遥か昔、この世界は「言葉の力」によって形作られた。

言葉は単なるコミュニケーションの手段ではなく、魔力の源であり、世界を動かす原動力であった。

偉大な言語学者たちは、英語と中国語という二大言語体系の力を解明し、人々は言葉を学ぶことで魔法を使えるようになった。

しかし、時代の流れとともに、言葉の真の力は忘れ去られ、人々は言葉を軽んじるようになった。

そして今、言葉の力が失われつつあるこの世界に、新たな旅人が現れた...
