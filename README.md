# 語源の旅人 - Dragon Quest Style RPG

ドラゴンクエスト風の本格RPGで英語・中国語を学ぼう！

## ゲーム概要

「語源の旅人」は、ドラゴンクエスト風のコマンド選択式RPGです。
敵の弱点である言語（英語・中国語）を使って攻撃すると、ダメージが増加し、
言語スキルも成長します。

## 機能

### ゲームシステム
- 🎮 **フィールド移動**: タイルマップ方式、矢印キー/タッチ操作対応
- ⚔️ **ランダムエンカウント**: フィールドやダンジョンで敵と遭遇
- 🎯 **コマンド選択式戦闘**: 戦う/特技/道具/逃げる
- 📈 **成長システム**: レベルアップ、ステータス上昇、スキル習得
- 🛡️ **装備システム**: 武器・防具の装備変更
- 💾 **セーブ機能**: 宿屋でセーブ可能

### 言語学習
- 🇬🇧 **英語スキル**: 英語レベルに応じてスキル習得
- 🇨🇳 **中国語スキル**: 中国語レベルに応じてスキル習得
- 📚 **単語帳**: 戦闘で覚えた単語を確認可能
- 🎯 **弱点システム**: 敵の弱点言語で攻撃するとダメージUP

### ワールド
- 🏘️ **村**: セーブ・ショップ・宿屋
- 🌲 **フィールド**: ランダムエンカウント
- 🏰 **ダンジョン**: 強敵が出現

## 操作方法

### PC（キーボード）
- **方向キー/WASD**: 移動
- **スペース/Enter**: 決定/調べる
- **Escape/Backspace**: キャンセル/メニュー

### モバイル（タッチ）
- **画面スワイプ**: 移動
- **Aボタン**: 決定/調べる
- **Bボタン**: キャンセル/メニュー

## 技術仕様

- **Canvas API**: ゲーム描画
- **Web Audio API**: サウンド生成（8bit風）
- **LocalStorage**: セーブデータ保存
- **レスポンシブ**: PC/モバイル対応

## 遊び方

1. `index.html` をブラウザで開く
2. 「はじめから」を選択
3. 村長に話しかけてヒントを得る
4. フィールドへ出てモンスターと戦う
5. 英語・中国語スキルを使って弱点を突く
6. 宿屋で休んでセーブ

## 開発

```bash
# ローカルサーバー起動（例）
python3 -m http.server 8000

# ブラウザで開く
open http://localhost:8000
```
